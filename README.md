# [Messenger](https://messenger-imkopyova.netlify.app/)
Messenger – приложение для обмена сообщениями

### Спринт 3
Pull request – https://github.com/imkopyova/middle.messenger.praktikum.yandex/pull/2

### Функционал
- авторизация
- изменение данных пользователя
- создание чатов

## Запуск
В режиме разработки:
```bash
npm run start
```
Тесты:
```bash
npm run test
```
