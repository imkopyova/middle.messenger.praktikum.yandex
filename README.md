# [Messenger](https://imkopyova-messenger.herokuapp.com/)
Messenger – приложение для обмена сообщениями
https://imkopyova-messenger.herokuapp.com/

### Функционал
- авторизация
- изменение данных пользователя
- создание чатов

## Запуск
В режиме разработки:
```bash
npm run start
```
Тесты:
```bash
npm run test
```
